## ![alt text](https://fts2020-api.herokuapp.com/assets/screen/logo_64.png) FTS2020 Mobile

-   Create flats (as groups of people)
-   Create Tasks like "weekly cleaning"
-   Create schedule for tasks & keep track of your turn
